# Setup

Make all paths exist

```
mkdir -p ~/.config/mutt/cache/headers
mkdir -p ~/.config/mutt/cache/bodies
touch  ~/.config/mutt/certificates
mkdir -p  ~/.mbox
```

## Problem: /Users/X/.mbox: No such file or directory (errno = 2)

Solution: `mkdir -p  ~/.mbox`
