Text objects:
- `ih` for git hunk
- `ai/ii/aI/iI for` block of indented lines
- `al/il` for current line
- `i,/a,` for function parameter
- ` iv/av` change variable segments
- `au/iu` for url
- `ae/ie` for entire buffer
- `ib/ab` for Quotes, Parenthesis and braces
Ctags:
- `^]` first match
- `:tn` next match
- `:tp` previous match
